# Premium Calculator - Security & Privacy Policy

## 1. DATA COLLECTION & PRIVACY

### What We Collect
- **NO personal data collection**
- **NO user tracking**
- **NO analytics**
- **NO ads**
- **NO third-party services**

### Local Storage
- Calculation history stored locally on your device only
- Configuration file stored in app's private directory
- No data transmitted to any server

### Permissions
The app requests minimal permissions:
- **INTERNET**: Not used (reserved for future use)
- **ACCESS_NETWORK_STATE**: Not used

## 2. SECURITY FEATURES

### Input Validation
✓ All calculations are validated  
✓ Injection attacks are prevented  
✓ Buffer overflow protection  
✓ Safe evaluation environment with restricted namespace

### Code Security
✓ No hardcoded secrets
✓ No external API calls
✓ No third-party libraries with vulnerabilities
✓ Regular security audits

### App Signing
✓ APK is cryptographically signed
✓ 4096-bit RSA key
✓ Valid for 100 years

## 3. PERMISSIONS JUSTIFICATION

## 4. DATA DELETION

Users can delete all app data by:
1. Settings > Apps > Premium Calculator > Clear Data
2. This removes all calculation history and preferences
3. This action is permanent and cannot be undone
